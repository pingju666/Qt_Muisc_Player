
# 凭摆的音乐播放器 🎵

一个基于 Qt 框架开发的桌面音乐播放器，兼具美观的二次元界面与实用的播放功能。

## ✨ 功能特性

- 🎨 **二次元风格界面**：采用动漫插画作为背景，视觉体验独特
- 🔍 **歌曲搜索**：支持快速检索本地音乐文件
- 📝 **实时歌词显示**：自动匹配并滚动显示歌词
- 🎛️ **完整播放控制**：
  - 上一首 / 下一首切换
  - 播放 / 暂停
  - 播放列表管理
  - 多种播放模式（顺序、单曲循环、随机等）
  - 音量调节
- 📁 **本地文件导入**：支持从文件夹添加音乐

## 🛠️ 技术栈

- **框架**：Qt (C++)
- **UI设计**：自定义QSS样式表，实现二次元风格界面
- **音频播放**：Qt Multimedia 模块
